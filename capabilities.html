Accessibility at the Edge Community Group

Overlay Capabilities Inventory

This document describes some functionalities that can be provided "at the edge", and attempts to answer several questions about them.

The questions are:

  • [2] Locating functionality in the stack to achieve maximum benefit: Source Code Perspective
  • Is this functionality duplicated elsewhere in the stack?
  • What is the benefit of doing this at the edge?
  • Related topics

Inventory

User-triggered interaction/opt-in

User opt-ins via interactive. Personalization. The end-user can turn on those enhancements that help them

[2] Sites may offer, at source code level, user-triggered interactions, rather than offering them in a JavaScript layer.

Redundancy:
Many enhancements such as contrast are available on the operating system level.

Runs automatically, no user action needed

Overlay capabilities that run on the page, without any action needed by the end-user.

Portable cross-site profile expressing accessibility-oriented preferences

Portable profile expressing accessibility-oriented preferences, that can be presented by the end-user (via the overlay) to tailor a website's experience to the user's preferences.

Automated Detection

Algorithmic detection of violations of WCAG 2.x success critera.

Redundancy:
In-browser scanners, online scanners perform detection. A large number of companies do this too for a fee.
Related Topics:
False positives, false negatives; certainty

Automated Remediation

Remediation of a violation of a WCAG 2.x success critera by programmatic means.

Redundancy:
Remediating at source, by hand
Related Topics:
False positives, false negatives

Notifications

Messages shown to the end-user to alert the user to an event, information or a state.

Related Topics:
What is required to disclose to the user?

Human in the loop on automated remediation suggestions

Overlay owner is asked to accept programmatic suggestions. This enables democratization of accessibility tasks and acceleration.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Accessibility provider visibly displayed

Display to website users the name of the primary accessibility provider, who stands behind the accessibility of the site, offering transparency. This may be part of the accessibility statement.

[2] The provider can be listed in the source text.

Redundancy:
Not redundant: an innovation unique to overlays of this type. Sometimes (rarely) the accessibilty provider is mentioned in the accessibility statement or accessibility policy.

Accessibility statement access

A link to the site's accessibility statement

[2] The link can be hosted on the source page.

Redundancy:
The link in the footer of the site

Accessibility issue reporting

A form, or email link, enabling end-users to report barriers to accessibility to the website owner

Redundancy:
Contact us in the accessibility statement
Related Topics:
Responsibility for responding. Requires that the company receiving the report has expertise to respond.

Profile enabling a bundle of user-triggered interactions/opt-ins (on a site)

Optional profiles that group interactive features together into bundles, making it easier to turn on groups of features, e.g. high contrast and larger text size.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Accessibility editing, by site owner

An interface in the overlay, available to authorized users, enabling manual editing of code that affects accessibility, e.g. alt tags for images, or ARIA labels.

Redundancy:
Not redundant: an innovation unique to overlays of this type. Provides an alternative to standard methods of coding the site, either via source code, web platform templates/CSS adjustments, etc.
Related Topics:
This editing needs to be done by people with expertise. Can be a source of information for a subject matter expert.

Accessibility editing, by 3rd party

A third party, via the overlay, codes the accessibility of the site.

Redundancy:
Not redundant: an innovation unique to overlays of this type. Without an overlay, this could be an outsourced remediation team that has login authorization to the source code and/or admin of the CMS/Platform.

Change Text Size

Increase or decrease font size, by clicking an interactive widget.

[2] Source code specifies text size as per the original target audience and design, but only for content coming from the source.

Redundancy:
Text sizes are configurable in the operating system. Not clear if this propagates across all dynamically loaded content.
Benefit of being on the Edge:
Learning how to make such changes in browser and/or operating system configurations is challenging for many, and particularly challenging for the communities that most need it. Offering overlay access raises awareness that such capabilities are present and does not prevent users from learning how to configure it in other places. Overlays are site-specific, making such accomodations on the sites where they are requested. An overlay could propagate such preferences across sites, browsers, operating systems and/or devices, to manage the complete experience including injected and/or dynamic content.

Change Text Spacing

Modify text and line spacing for improved readability for dyslexic and visually impaired users. Offers three degrees of adjustments for a personalized and more accessible reading experience. ,

Redundancy:
Defaults are configurable in the operating system.

Change Cursor Size

Increases the standard cursor size by 400% to ensure the pointer always remains in sight. Allows for faster and more accessible navigation through hyperlinks, tabs. and form elements.

Redundancy:
Defaults are configurable in the operating system.

Change Link Highlights Visibility

Selecting the text or image that serves as a hyperlink and causing it to be displayed in a different, more visible way, to help identify the links on a page more easily.

Redundancy:
Defaults are configurable in the browser.

Pause Animations

The ability to stop animations or other moving elements on a web page from playing.

Redundancy:
Some players offer a pause. Context menu or developer tools.

Apply Light Contrast, Dark Contrast, Invert Colors, Color Desaturation

Color modifications that are useful to people with visual impairments. They modify the difference between the lightest and darkest parts of a web page, reverse colors, or reduce theior intensity.

Redundancy:
Defaults are configurable in the operating system.

Tooltips

small pop-up window that appears on hover to display additional information about an element on the page, providing context or clarification.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Reading Guide

A horizontal line that appears below text to help the reader keep their place as they read, follow along with the text and stay focused.

Redundancy:
Not redundant: an innovation unique to overlays of this type. There may be browser extensions that provide this functionality.

Accessible Fonts

Fonts selected for their legibilitiy, that are generally found to be easier to read, especially for people with low vision or reading difficulties.

Redundancy:
Defaults are configurable in the operating system.

Dyslexia Friendly Font

Fonts considered to be more dyslexia-friendly as they have more distinctive symbols that avoid confusion, increased baselines and thinner tops, and a larger x-height.

Redundancy:
Fonts can be configured in the operating system.

Page Structure Show Headings, Links, Landmarks

ALternative navigation methods that are commonly used by screen readers, made available to the sighted user. These include the ability to browse a page by its headings, which can be helpful for understanding the overall structure of the page; links; buttons.

Redundancy:
Available in screen readers

Reset All

One button click to reset any user triggered enhancement and restore the page to its default state.

Redundancy:
No need.

Screen Reader

On page screen reader

Redundancy:
Screen reader installed at the operating system level and used as a standalone application. Also (less common) screen readers in an application such as a browser.

Keyboard Navigation

Enhance visual indicators that support keyboard navigation such as underlines or color changes; optionally provide other assistance such as mapping interactive elements to keyboard shortcuts.

Redundancy:
Not redundant: an innovation unique to overlays of this type. Some aspects can be enhanced in the operating system or browser configuration, such as focus indication.

Image Alts

Short piece of text associated with an image on a web page. It is intended to provide a description of the image for users who cannot see it, such as those using screen readers or text-only browsers.

Redundancy:
Some computer vision image identitification is included in some browser extensions.

Form Labels

A label associated with each form field accurately describing its purpose. The label should be consistent with the logical structure of the form overall.

[2] Code with proper form labels.

Redundancy:
Not redundant: an innovation unique to overlays of this type.
Benefit of being on the Edge:
Apparently, after twenty years of consistent effort at changing how we train designers and developers of forms, only a subset are doing it right today, but we still need the labels. Having an overlay provide them escalates the need, hopefully attracts dev attention and gets the resource allocated. Certainly, an overlay does not prevent source from being fixed, when it can be fixed (e.g. there are some legacy codebases where fixing in the edge is preferred).

Dictionary

Provide definitions of words or phrases that may be unfamiliar to some users; provide pronunciation guidance; provide synonyms which can help users with cognitive disabilities better comprehend the content.

Redundancy:
Browsers and operating systems offer related features, generally on "right-click".

Form Validation

Ensure that screen reader users and people with disabilities receive clear feedback on the status of the form along with concise error messages. This includes the colors used, the proximity to the form field, and the time-sensitive nature of the screen reader announcement.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Interactive Elements

Ensure there are concise instructions and labels that are clear to sighted, low vision and non sighted users, and that the elements can be controlled with keyboard navigation. Provide such feedback on the status of the element and any changes that occur.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Autoplay Videos

Provide ability to pause, mute and/or stop the video; make these controls keyboard navigation accessible.

[2] Videos should be made available with these capabilities, as they are built into HTML 5.

Redundancy:
Not redundant: an innovation unique to overlays of this type. Some sites offer the option.
Benefit of being on the Edge:
Apparently, after years of consistent effort at changing how we train designers and developers of websites that host video, only a subset are doing it right today, and we still need these controls. Having an overlay deliver these capabilities is not a problem for end users, it is more service for users. The overlay may escalate the need, hopefully attracting dev attention and encourage better source development in the future. Certainly, an overlay does not prevent source from being fixed, when it can be fixed (e.g. there are some legacy codebases where fixing in the edge is preferred).

External Link Targets

Assure the link is announced correctly as a link, and inform users if it opens in a new tab or window. In addition, assess broken links: verify that the link resolves, and mark broken links as broken.

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Ambiguous Links

Provide descriptive and meaningful link text that accurately reflects the purpose or destination of the link, particularly needed for link text such as "click here" or "more."

Redundancy:
Not redundant: an innovation unique to overlays of this type.

Broken Links

Assess if the link resolves, and mark broken links as broken.

Redundancy:
Not redundant: an innovation unique to overlays of this type. There are some browser extensions that perform some of this functionality.

Video Captioning

Displaying in text along with video, synchronized with the video, what is heard.

Missing Titles

Meaningful titles, for pages and/or iFrames within pages, allow screen reader users to browse by means of titles.

Element Focus

Visible indication when an interactive element on the page is selected.

Site Language

Specify the primary language of a web page.

PDF Remediation

Assuring that a PDF is tagged and marked up sufficiently so that it can be accessed and understood by people with disabilities, including those who use assistive technology such as a screen reader.

Content Moderation

Assuring that the content on a site meets content policy. The default policy includes everyday phrases with racist origins, and gendered or exclusive expressions and terms that are deemed to exclude audiences.

Text Align

Text alignment enhances readability. The inconsistent spacing between words created by justified text can be problematic for people with cognitive concerns such as Dyslexia. Users can determine the alignment best suited to their vision.

Line Height

Increase or decrease line height to distinguish between lines of text.

Accessible Pricing

Strikethrough pricing is a visual representation of the original price of a product in a strikethrough or crossed out font, and then an associatec new price next to it. Screen readers read the struckout price like any other number, so ARIA markup needs to be added to indicate that the struck out price is the old, no longer valid price.

Live Site Translations

Offer a user-triggered method to change the language of a web page in real time, by translating the current page into a language selected by the user.

Smart Contrast

The colors of text and background that does not meet minimum contrast requirements are adjusted in such a way that the minimum change is made. The advantage of such minimal changes is that the colors will resemble, or remain close to, the original brand design intention.
